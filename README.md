# AcireAutoDownload

> Antes de iniciar o aplicativo, é necessário configurar o servidor.

## Como configurar o servidor

### Passo 1: Instalar o XAMPP
Baixe e instale o XAMPP para ter o Apache e o MySQL funcionando no seu computador.

### Passo 2: Configurar o MySQL para iniciar automaticamente
Execute o arquivo `mysql_service_setup.bat` para que o MySQL inicie automaticamente junto com o Windows.

### Passo 3: Instalar as tabelas do banco de dados
Clique e execute o arquivo `install_database.bat` para instalar todas as tabelas do MySQL.

> Após completar esses passos, o servidor estará pronto e você poderá iniciar o aplicativo.
